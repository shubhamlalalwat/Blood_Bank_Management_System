Title - Blood Donation Management System

Technologies and Tools:
● OS – Windows 10
● Front End –JSP
● Database – MySQL server
● Scripting Language- JavaScript, CSS
● Browser– Any browser compatible with IE 5.0 or onwards
● Server – Apache Tomcat (any version)
● Tools- Eclipse-IDE 2020-12

The process to Run the project:- 
1) Start eclipse IDE-2020-12.
2) Open Blood Bank Management system folder in eclipse.
3) Goto the web content folder.
4) click and run the index.JSP page.
5) Enter all details and click to submit.
6) Now move to the Admin Login page. Username - shubham and password - lala and click to submit.
7) Now you can add a donor to donate blood and save.
